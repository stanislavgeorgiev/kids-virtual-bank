## How to Create Screenshots

### For Google Sheets
1. Open your Kids Virtual Bank spreadsheet
2. Navigate to the sheet you want to capture
3. Use screenshot tool:
   - **Mac**: Cmd + Shift + 4, then spacebar (captures window)
   - **Windows**: Snipping Tool or Win + Shift + S
   - **Chrome**: Built-in screenshot tool
4. Save with descriptive name
5. Add to this folder

### For Forms
1. Open form in preview mode (eye icon)
2. Take screenshot of the form
3. Consider capturing both desktop and mobile views

### For Apps Script
1. Open Apps Script editor
2. Capture relevant views (code, triggers, executions)
3. Annotate if needed to highlight important areas

## Tips for Good Screenshots

- Use consistent size/format (PNG recommended)
- Crop to relevant content
- Blur sensitive information (real kid names, etc.)
- Use annotations to highlight important features
- Consider creating a multi-panel comparison image
